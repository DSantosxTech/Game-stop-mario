# Stop-Mario Matemático

Jogo de STOP com tema do Mario e desafios matemáticos.

- *Níveis de Dificuldade:*
  - Fácil: Números de 10 a 50.
  - Médio: Números de 51 a 100.
  - Difícil: Números de 101 a 200.
